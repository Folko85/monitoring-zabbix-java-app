Нужно настроить мониторинг java + spring boot + actuator в забикс

Требуется:
1) Сделать стенд для java приложения и zabbix
   - Развертывание: Docker
   - Версия Zabbix: 6.4.20
   - Требования к портам: нет особых требований
2) Мониторинг будет сделан с помощью zabbix agent на машине с java приложения
   - Spring Boot Actuator: все endpoints доступны
   - Порт приложения: стандартный (8080)
3) Нужен метод для быстрой настройки zabbix агента для получения метрик
   - Автоматизация: Ansible
4) Собираемые метрики:
   - Все метрики Actuator (JVM и application метрики)
5) Конфигурация Zabbix:
   - Items для сбора метрик
   - Dashboards для визуализации
